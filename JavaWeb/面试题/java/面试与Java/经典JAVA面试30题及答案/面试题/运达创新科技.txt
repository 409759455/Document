单位简介... 
西南交通大学智能控制与仿真工程中心是铁道部于2000年2月批准的部级开放式工程研究中心。
中心目前拥有研究开发人员31人，具有博士和硕士学位的人数占总人数的一半以上。
在“211”工程建设中，中心已拥有先进的研究及开发设备。
以中心为基础于1992年10月组建的成都运达创新科技有限公司，是具备高科技产品开发与生产制造能力的科技型企业。
公司已于2002年1月通过ISO9001:2000认证。 
列车模拟器是公司的主要产品之一，已生产一百四十余台在全路推广应用，在国内铁路行业有较大影响。
近期的研发项目
铁道部科技发展项目“提速列车优化操纵技术与模拟器的开发与应用”；
城市轨道交通模拟驾驶装置及司机培训系统；
网络化、交互式计算机辅助教学系统；
非正常行车条件下行车办法教学软件；
电力机车耗电量及操纵评价系统；
机车检修质量控制系统；
车载无线数据传输系统；
车载计算机控制与检测系统。
