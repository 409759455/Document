应试人员面谈记录表

　　主试人员： 应试人员： 应聘职位： 应试时间： 年 月 日 地点：

　　项目参考内容记录评价

　　礼仪风度仪容、衣着；行为、举止；敲门、走路、坐姿、站立等的仪态；口语是否文明等。

　　求职动机

　　1. 你是怎么了解到我们的招聘信息？

　　2. 你了解高奇企业吗？你对高奇企业最感兴趣的是什么？

　　3. 你对于这个职位最感兴趣的是什么？

　　4. 对这个职位，你最希望得到的激励是什么？

　　5. 如果你被录用，你要做的第一件事是什么？

　　社交能力、人际关系、社会资源

　　1. 你交朋友最注重什么？

　　2. 请您介绍你的家庭？

　　3. 你的家人/朋友/同事/上司如何评价你？

　　4. 你希望在什么样的领导下工作？

　　5. 谈谈你如何领导一个团队？

　　6. 你愿意独立工作还是同其他人一起合作？

　　7. 你在工作中如何与同事相处？

　　8. 在公司中你一般会参加什么活动？

　　9. 你有哪些社会资源和社会关系？

　　判断力、情绪稳定性（对突发事件和难堪事件的反应）

　　1. 你在压力下如何工作？

　　2. 你一般如何应对冲突？

　　3. 谈谈你如何有效的管理时间？

　　4. 《谁动了我的奶酪》这本书你看过吗，你怎样应付变化？

　　5. 假如A公司与B公司同时录用了你，你将如何？

　　6. 你在工作上没有多少经验，你打算怎样学习必备的知识？

　　7. 你经常换工作，我们怎么知道这份工作你会坚持多久呢？

　　8. 如果你没有被我司录用，你会怎么做？

　　个人品质

　　1. 请谈谈你自己的性格特征、优缺点？

　　2. 谈谈你的兴趣爱好，下班以后喜欢做什么？

　　3. 据你自我分析，最适合你的工作是什么？

　　4. 你认为一个人最重要的品质是什么？

　　5. 如果你可以改变你性格中的一件事，那是什么？为什么？

　　6. 你能否“受人之托忠人之事”？

　　7. 得过什么奖励？受过什么处罚？原因？

　　8. 你自认为你比我面试的其他应征者强在什么地方？

　　9. 你如何处理工作中遇到的批评和指责？

　　10. 你如何处理工作中你认为不对的事情？

　　责任心、纪律性

　　1. 面试是否迟到？

　　2. 对公司的规章制度的看法是什么？

　　3. 你对委任的任务完成不了时如何处理？

　　4. 如果你离开目前的工作，公司会有什么情况发生？

　　5. 谈谈你对做一件事的结果和过程的看法，更看重结果还是过程？为什么？

　　6. 谈谈你对责任的认识？

　　工作稳定性

　　1. 什么原因离职？有什么不满意的地方？

　　2. 你在每家公司呆的时间都不是很长，为什么？

　　3. 离开原单位能提供离职证明吗？

　　4. 从现在到将来的5年时间内，你希望做些什么？

　　5. 你工作以来的薪资分别是多少？

　　6. 你期望目前这个职位的薪金是多少？

　　7. 对工作地点有什么要求？能出差吗？能驻外吗？

　　8. 你如何在职业和家庭之间维持平衡？

　　9. 什么时候可以正常上班？

　　专业技能知识（对专业知识的了解程度、工作对所要从事的工作的认识等）

　　1. 学历、专业是否达到要求？

　　2. 是否具备足够的工作经验？

　　3. 谈谈你对这个职位的认识？

　　4. 笔试成绩

　　5. 专业知识问答
